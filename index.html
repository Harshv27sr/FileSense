AI File Management System

AI File Management System is a smart desktop application developed using Python and Tkinter. It helps users manage files and folders efficiently by providing features like automatic file organization, duplicate file detection, file scanning, trash management, and database tracking.

The system improves productivity by reducing manual work and making file handling faster, safer, and more organized. It is specially useful for students, offices, and personal computer users who manage large numbers of files daily.

This project is developed as a Windows desktop application (.exe) and not as a website. The source code of the project is available on GitHub, and the executable application file is shared through Google Drive for direct use and testing.
